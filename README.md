# HR Analytics Dashboard

## Project Overview
This project analyzes employee attrition, workforce demographics, compensation, and retention trends using SQL, Excel, Power BI, and Tableau.

## Tools Used
- SQL
- Microsoft Excel
- Power BI
- Tableau

## Dataset
- 50,000 Employee Records

## Key KPIs
- Total Employees: 50,000
- Attrition Count: 25,105
- Active Employees: 24,895
- Attrition Rate: 50.21%
- Average Age: 39 Years
- Average Salary: $26,000
- Average Tenure: 10.8 Years

## Key Insights
- Research & Development has the highest attrition rate.
- Promotion stagnation increases employee turnover.
- High-income employees also show high attrition.
- Work-life balance impacts retention.
